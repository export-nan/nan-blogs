                              # 智谷后台管理项目

                              # B端管理部分

                              面向商家的管理系统部分，基于Ant Design Pro的B端管理系统

                              # 技术栈

                              vue3: https://vuejs.org/open in new window
                              Ant Design Pro: https://pro.antdv.com/docs/getting-startedopen in new window

                              # 后端部分

                              基于midway编写的node服务端系统

                              # 技术栈

                              Midway: http://www.midwayjs.org/open in new window 数据库: mysql

                              # 快速开始

                              # 环境配置

                              1. 安装node, 版本大于14V: https://nodejs.org/en/open in new window

                              2. 安装pnpm: https://pnpm.io/zh/installationopen in new window

                              3. 安装docker,(可选，主要用于搭建数据库环境): https://www.docker.com/open in new window

                              # 快速开始

                              1. 启动数据库（如果使用docker可以直接运行项目下develop中的docker-compose.yaml)
                              $ docker compose -f "develop/docker-compose.yaml" up -d --build
                              1. 安装依赖包
                              pnpm --filter service install
                              1. 运行项目
                              pnpm --filter service dev
